FindCoder
Présentation du projet :
FindCoder, une plateforme dédiée à la mise en relation entre les employeurs et les développeurs. Les développeurs peuvent créer un profil complet, ajoutant leurs compétences, CV, LinkedIn, GitHub et portfolio. Les employeurs peuvent parcourir ces profils, rechercher des talents et communiquer via un système de messagerie intégré. L'objectif est de simplifier le processus de recrutement pour les développeurs en facilitant la découverte des talents et en établissant des connexions directes.

Outils

Backend: Laravel (PHP) with API integration.

Frontend: Vue.js 3 for dynamic user interfaces.

Database: MySQL for storing user data and posts.

Authentication: Laravel Sanctum for secure user authentication.

Real-time Communication: Pusher for real-time chat between users.

Version Control: GitHub for code management and collaboration.
